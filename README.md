# MemoryGame

> **Status:** Archived — this is an older project kept for reference.

A simple “Find the Pair” memory game built with ASP.NET MVC, Entity Framework, and SQL Server.  
The project remains available online as a playable demo.

🎮 **Play online:** https://znajdzpare.mdwojak.pl/

## Features
- Multiple difficulty levels (progressive board sizes)
- Card flipping and pair matching
- Saving game results as a guest (no login required)
- Top 5 best results
- Last 5 played games
- Frontend built with JavaScript and jQuery

---

## Tech Stack
- **ASP.NET MVC (C#)**
- **Entity Framework**
- **SQL Server** (connection string in `web.config`)
- **JavaScript + jQuery**
- **Razor Views**
- **CSS**
